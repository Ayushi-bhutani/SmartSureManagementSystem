## 📞 Interview Questions & Answers

### Q: "Why did you choose Angular over React?"
**A:** "I chose Angular for its opinionated structure, built-in features like dependency injection and RxJS, and strong TypeScript support. For an enterprise application like SmartSure, Angular's comprehensive framework reduces decision fatigue and ensures consistency."

### Q: "How did you handle state management?"
**A:** "I used a service-based approach with RxJS BehaviorSubjects for shared state. For example, the NotificationService manages notification state across components, and the AuthService handles user authentication state."

### Q: "What was the most challenging feature?"
**A:** "The Razorpay payment integration was challenging because it required understanding the payment flow, handling callbacks, and ensuring security. I had to implement proper error handling and verify payment signatures on the backend."

### Q: "How did you ensure code quality?"
**A:** "I used TypeScript strict mode, followed SOLID principles, implemented proper error handling, and organized code into reusable services and components. I also used Angular's built-in features like guards and interceptors for cross-cutting concerns."

### Q: "How would you scale this application?"
**A:** "The microservices architecture already supports horizontal scaling. I would add caching (Redis), implement CDN for static assets, use database read replicas, and add load balancing. The frontend can be deployed to multiple regions."
