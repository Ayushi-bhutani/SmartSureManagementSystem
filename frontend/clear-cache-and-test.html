This tool will help you fix the admin login issue by clearing browser cache and testing the login flow.

⚠️ IMPORTANT: The issue is caused by browser caching old JavaScript files. Follow the steps below to fix it.

Step 1: Clear Browser Cache

Option A: Hard Refresh (Recommended)
  1. Close all SmartSure tabs in your browser
  2. Press Ctrl + Shift + Delete (Windows) or Cmd + Shift + Delete (Mac)
  3. Select "Cached images and files"
  4. Click "Clear data"
  5. OR simply press Ctrl + F5 on the login page
Option B: Clear SmartSure Data (Quick Fix)
  1. Click the button below to clear all SmartSure data from localStorage
  2. Then do a hard refresh with Ctrl + F5
