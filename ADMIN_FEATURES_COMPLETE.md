3. Toggle type status

5. View all insurance plans
6. Toggle plan status
7. Click edit/delete actions
```

```

   - Min Purchase: 10000
   - Max Discount: 5000
   - Usage Limit: 100
   - Dates: Select range
3. Click "Create Discount"
4. View in discounts table
5. Edit discount
6. Toggle status
7. Delete discount

6. Review selections
7. Click "Generate Report"

9. Download report
