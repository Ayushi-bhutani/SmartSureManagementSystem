sequenceDiagram actor User participant Angular participant Gateway participant Identity participant DB User->>Angular: Enter Credentials Angular->>Gateway: POST /gateway/auth/login Gateway->>Identity: POST /api/auth/login Identity->>DB: SELECT User WHERE Email DB-->>Identity: User Data Identity->>Identity: Verify Password (BCrypt) Identity->>Identity: Generate JWT Token
(Claims: Id, Email, Role) Identity-->>Gateway: JWT Token + User Info Gateway-->>Angular: Token Angular->>Angular: Store Token in LocalStorage Angular-->>User: Redirect to Dashboard Note over User,DB: Subsequent Requests User->>Angular: Access Protected Route Angular->>Angular: Add Authorization Header
Bearer {token} Angular->>Gateway: GET /gateway/policies
Authorization: Bearer {token} Gateway->>Gateway: Validate JWT Token
(Signature, Expiry, Claims) Gateway->>Policy: GET /api/policies Policy-->>Gateway: Policies Data Gateway-->>Angular: Response Angular-->>User: Display Data
